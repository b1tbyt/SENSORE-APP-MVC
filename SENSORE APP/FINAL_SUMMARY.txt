## ?? HOW IT WORKS

### Every 100ms Loop
1. **PressureDataService** generates new heatmap data
2. **OnHeatmapUpdated** event fires
3. **HeatmapBroadcastService** converts to JSON
4. **SignalR** broadcasts to all connected clients
5. **JavaScript** receives update
6. **Browser** updates 1,024 table cells
7. **CSS** smoothly transitions colors
8. Result: Seamless real-time animation

## ?? CUSTOMIZATION

### Change Colors
Edit `Index.cshtml` CSS section:
```css
.sens-card-header {
    background-color: #0050A0;  /* Change me */
}
```

### Change Update Speed
Edit `PressureDataService.cs`:
```csharp
await Task.Delay(100, cancellationToken);  // Change 100 to whatever
```

### Change Layout
Edit `Index.cshtml` HTML grid:
```html
<div class="col-md-8">...</div>  <!-- 60% width -->
<div class="col-md-4">...</div>  <!-- 40% width -->
```

## ?? TROUBLESHOOTING

| Issue | Fix |
|-------|-----|
| App won't run | Run `dotnet restore` first |
| Port already in use | Change port in `launchSettings.json` |
| No heatmap updates | Check console (F12) for errors |
| High CPU usage | Increase `Task.Delay()` to 200ms |
| Memory growing | Restart app every few hours |

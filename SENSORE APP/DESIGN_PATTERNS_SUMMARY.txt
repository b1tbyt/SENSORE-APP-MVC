?? FACTORY PATTERN (Creational)
?????????????????????????????????????????????????????????????????????????????

Location: Services/Factories/MessageFactory.cs

Interfaces:
  ? IMessageFactory
  ? AlertMessageFactory

Features:
  ? Centralizes message creation
  ? Input validation at creation time
  ? Supports multiple message types
  ? Extensible for new factories

Usage:
  var message = _factory.CreateMessage("Patient", "Text");
  var alert = _alertFactory.CreatePressureAlert(240);

Benefits:
  ? Encapsulation
  ? Easy to test
  ? Single responsibility
  ? Reduced coupling

?? ADAPTER PATTERN (Structural)
?????????????????????????????????????????????????????????????????????????????

Location: Services/Adapters/PressureSensorAdapter.cs

Interfaces:
  ? ILegacyPressureSensor (External/Legacy)
  ? IStandardPressureSensor (Our Standard)

Adapters:
  ? LegacyPressureSensorAdapter
  ? ThirdPartySensorAdapter

Features:
  ? Converts 0-1000 scale ? 0-255 scale
  ? Resizes any dimension ? 32×32
  ? Applies calibration factors
  ? Provides quality scoring

Usage:

    new LegacyPressureSensorAdapter(legacySensor);
  byte[,] data = adapted.GetNormalizedReadings();

Benefits:
  ? Legacy system integration
  ? No modification to existing code
  ? Clean abstraction
  ? Format normalization

?? STRATEGY PATTERN (Behavioral)
?????????????????????????????????????????????????????????????????????????????

Location: Services/Strategies/RiskAnalysisStrategy.cs

Interface:
  ? IRiskAnalysisStrategy

Concrete Strategies:
  ? PeakPressureStrategy
  ? ContactAreaStrategy
  ? HotspotConcentrationStrategy
  ? PressureDistributionStrategy

Context:
  ? StrategyBasedRiskAnalyzer

Features:
  ? Runtime algorithm switching
  ? Multi-strategy consensus
  ? Intervention detection
  ? Easily extensible

Usage:
  _analyzer.SetStrategy(new PeakPressureStrategy());
  var result = _analyzer.Analyze(heatmap);
  
  // Or multi-strategy:
  var consensus = _analyzer.AnalyzeWithAllStrategies(heatmap);

Benefits:
  ? No conditional logic
  ? Easy to add new algorithms
  ? Runtime flexibility
  ? Clean separation

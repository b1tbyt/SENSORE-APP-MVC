## ?? HOW TO USE EACH BUTTON

### 1. Change Data Timeframe
```
Step 1: Click 1h, 6h, 24h, or 7d
Step 2: Chart updates automatically
Step 3: Metrics refresh
Result: See different time periods
```

### 2. Log Reposition
```
Step 1: Click "Log Reposition" button
Step 2: See "Logged!" feedback
Step 3: Get confirmation message
Step 4: Timer resets to 30 minutes
Result: Action recorded in system
```

### 3. Contact Support
```
Step 1: Click "Contact Support" button
Step 2: Modal opens with 3 options
Step 3: Choose: Call Ģ Email Ģ Message
Result: Get support via preferred method
```

### 4. Send Support Message
```
Step 1: Click "Send Message" in modal
Step 2: Fill Subject field
Step 3: Fill Message field
Step 4: Select Priority (Low/Medium/High)
Step 5: Click "Send Message" button
Step 6: See confirmation
Result: Support ticket submitted
```

### 5. View FAQs
```
Step 1: Click "View FAQs" button
Step 2: Modal opens with accordion
Step 3: Click question to expand
Step 4: Read answer
Step 5: Click other questions
Result: Get answers to common questions
```

### 6. Generate Report
```
Step 1: Click "Generate Personal Report"
Step 2: Report loads with data
Step 3: See peak/average pressure
Step 4: See risk score & recommendations
Result: Report ready to review/download
```

### 7. Download Report
```
Step 1: Click "Download as PDF" or "CSV"
Step 2: Select location for save
Step 3: File downloads
Result: Have report file on computer
```

### 8. Send Message (Chat)
```
Step 1: Type message in input field
Step 2: Click "Send" button
Step 3: Message appears in thread
Step 4: See confirmation
Result: Message sent to clinician
```

### 9. Submit Note
```
Step 1: Optionally select timestamp
Step 2: Type note in textarea
Step 3: Click "Submit Note"
Step 4: See confirmation
Result: Note added to record
```

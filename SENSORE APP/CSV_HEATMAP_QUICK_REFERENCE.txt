## ?? CSV FILE FORMAT

```
32 rows × 32 columns
Comma-separated values
Each value: 0-255 (byte)

Example:
10,15,20,18,22,25,...,80
12,17,22,20,24,27,...,82
...
(32 rows total)
```

## ?? NEW ENDPOINTS

```
GET  /Patient/GetAvailableHeatmaps
     Returns: { heatmaps: [...], count: N }

POST /Patient/LoadHeatmapFromCsv?fileName=file.csv
     Loads specific CSV

POST /Patient/LoadHeatmapFromCsv
     Loads random CSV

GET  /Patient/HeatmapManagement
     Returns management info
```
